# WeWork

> 最最最简单易用的企业微信SDK

https://pithyone.github.io/wechat/

## Requirement

- PHP >= 7.1.3
- SimpleXML PHP Extension
- DOM PHP Extension
- JSON PHP Extension
- OpenSSL PHP Extension

## Installation

```bash
composer require pithyone/wechat
```
